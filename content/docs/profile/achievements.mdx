# Achievements

Gamification helps keep you motivated. Here are the badges you can earn.

## Badge List

*   **Bug Hunter**: Fix 50 compiler errors.
*   **Week Warrior**: Maintain a 7-day streak.
*   **Night Owl**: Code between 12 AM and 4 AM.
*   **Polyglot**: Complete courses in 3 different languages.
