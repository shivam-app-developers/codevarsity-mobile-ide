### 📖 How It Works (Step-by-Step)

```
Step 1 — Search or Paste
Open the YouTube Hub tab. Search for any topic (e.g., "Python for
beginners") or paste a direct YouTube playlist/video URL.

Step 2 — Open the Workspace
Tap any result. The video opens on the top half of your screen
and a fully functional code editor opens on the bottom half.

Step 3 — Code Along in Real-Time
As the instructor explains concepts, type the code yourself in
the editor below. Hit "Run" to see the output instantly.

Step 4 — Save & Resume
Your progress is automatically saved. Come back any time and
continue from where you left off.
```
