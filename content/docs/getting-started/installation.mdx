# Installation & Setup

CoderKit is designed to be a mobile-first coding environment. Currently, it is available for Android devices.

## Android Installation

1.  Open the **Google Play Store** on your Android device.
2.  Search for **"CoderKit"**.
3.  Tap **Install**.
4.  Once installed, open the app.

## System Requirements

-   **Android Version:** Android 7.0 (Nougat) or higher.
-   **RAM:** Minimum 2GB (4GB recommended for smoother visualizer performance).
-   **Storage:** ~150MB for the base app. Additional space required for courses and offline compiler resources.

## Permissions

CoderKit may request the following permissions:

-   **Storage:** To save your code files locally on your device.
-   **Internet:** To sync your profile stats and download new courses. (Compilers run offline).
