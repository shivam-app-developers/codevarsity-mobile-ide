## What is CodeVarsity?

**CodeVarsity** is a professional mobile IDE and coding education platform that lets you **write, compile, and run real code offline** on your Android device. Unlike cloud-based coding apps, CodeVarsity runs everything locally - no internet required.

## Supported Languages

### Python 3.10 (via Chaquopy)

- **Full offline execution** with interactive `input()` support
- **Pre-installed libraries**: NumPy, Pandas, SciPy, Scikit-learn, Flask, Django, Requests, BeautifulSoup
- **Web Development**: Flask & Django with Waitress WSGI server

### Java (via ECJ Compiler)

- Compile and run Java 8-21 code offline
- Support for Generics, Lambdas, Streams
- Built-in HTTP server for web development

### Go (via Yaegi Interpreter)

- Full Go standard library support
- Goroutines and concurrency
- Real-time streaming output

### Web Development

- **HTML/CSS**: Live preview with linting and formatting
- **JavaScript**: ES6+ support via Mozilla Rhino
- **React & Vue.js**: In-browser compilation via Babel

### Additional Languages

- **SQL** (SQLite) - Query databases directly
- **Groovy** - Scripting and automation
- **Clojure** - Functional programming

## FAQ

### Can I run Python offline on Android?

**Yes!** CodeVarsity includes a full Python 3.10 runtime via Chaquopy. No internet required. Supports NumPy, Pandas, Flask, Django, and more.

### Does it support interactive input?

**Yes!** Use `input()` in Python, `Scanner` in Java, or `fmt.Scanln()` in Go for interactive CLI programs.

### Can I build web pages on my phone?

**Yes!** Write HTML, CSS, and JavaScript with live preview. Even use React or Vue.js with in-browser compilation.

### Is there algorithm visualization?

**Yes!** The Algorithmic Sandbox visualizes sorting algorithms (Bubble, Quick, Merge), searching, and graph algorithms (BFS, DFS, Dijkstra).

### Can I use it for data science?

**Yes!** Python comes with NumPy, Pandas, SciPy, and Scikit-learn pre-installed. Run real machine learning code on your phone.
