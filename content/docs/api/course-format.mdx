# Course Data Format

CoderKit courses are structured as JSON objects. This allows for dynamic updates and cross-platform compatibility.

## Structure

```json
{
  "id": "py-101",
  "title": "Python Basics",
  "modules": [
    {
      "id": "m1",
      "title": "Variables",
      "lessons": []
    }
  ]
}
```
